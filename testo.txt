# Firebase configurazione
In enviroment/enviroment.ts copio il json del mio firebase riguardamte l'app

# Firebase
Check the version:  firebase --version
Login to Firebase: firebase login
Start: firebase init e scelgo Hosting, scelgo il mio progetto spesa
Vengono creati 2 file  firebase.json e .firebaserc
dopo aver modificato firebase.json come dist   si procede con la pubblicazione:
1) ng build --prod
2) firebase deploy
